# Server Monitoring Data Engineering Pipeline

This project demonstrates an end-to-end data engineering pipeline for monitoring server performance.

## Architecture

Raw Server Logs
→ Azure Data Lake
→ Azure Data Factory
→ Databricks (Data Transformation)
→ Transformed Dataset
→ Power BI Dashboard

## Technologies Used

- Azure Data Lake Storage
- Azure Data Factory
- Azure Databricks
- PySpark
- Power BI

## Dashboard Features

- Server health monitoring
- CPU utilization analysis
- Downtime tracking
- Network traffic analysis
- System availability KPI

## Project Output

The pipeline processes server logs and generates analytical datasets used for a monitoring dashboard.
